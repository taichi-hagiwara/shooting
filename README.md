#　白黒シューティング


# 説明
Python と Pygame を使って作成されたグリッドベースのシューティングゲームです。次々と湧き出る敵を避けながら生き延びることを目標としたゲームで、弾を撃って黒いタイルを白にし、敵やタイルの変化に対処します。

## 操作方法と特徴
- プレイヤーは `W`、`A`、`S`、`D` キーで上下左右に移動できます。
- 白色のマスには入ることができますが、黒色のマスには入ることができません
- 矢印キー（`↑`、`↓`、`←`、`→`）を使って上下左右に弾を発射可能。
- 弾が黒マスに命中すると白マスに変化します
- 画面端から敵が登場し、ランダムに指定された目標タイルに向かって移動します。
- プレイヤーは青の円，敵は赤の四角，弾はグレーの円でそれぞれ表されます
- 黒色のマスは一定確立で「弾入りマス」に変化します。「弾入りマス」を踏むと弾を補充。
- 60秒間生き残ることを目指します。  
  - 生き延びると勝利（クリア画面）。
  ![クリア画面](/img/img_gameclear.png)
  - 敵とぶつかるとゲームオーバー(ゲームオーバー画面)。
  ![ゲームオーバー画面](/img/img_gameover.png)

- 「時間」と「弾数」などの情報は画面下部に表示されます。
- ゲームオーバー/ゲームクリア画面では，Rキーを押すとコンティニュー，Ｑキーを押すとゲームを完全に終了します
